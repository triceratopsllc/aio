# AIO（AI Overview対策）1ページ提案書

**対象:** 自社サイトが生成AI（ChatGPT / Claude / Perplexity / Google AI Overview）に正しく認識されていない / 引用されていないと感じている企業様

## 提供価値

生成AI時代の新しい流入ソース「AI検索」から、**指名引用されるサイト**にアップグレードします。

### 現状の課題

- Google 検索結果の上部が AI Overview に置き換わり、従来の SEO 施策だけではクリック流入が減衰
- ChatGPT / Claude / Perplexity の回答内で、自社より競合が引用されている
- サイトに構造化データ（JSON-LD）が無く、AIが内容を正確に要約できない
- robots.txt で AI クローラーをブロックしている（または未設定）ため、そもそも学習対象外

### 施策内容（Phase 1 基盤整備）

1. **JSON-LD 構造化データ導入** — 7スキーマ（Organization / FAQ / Service / HowTo / Article / BreadcrumbList / WebSite）を全ページに埋め込み
2. **AI クローラー許可設定** — GPTBot / ClaudeBot / PerplexityBot / Google-Extended など 19種を明示許可
3. **サイトマップ最適化** — 動的ページ含む全URLを網羅
4. **OGP / メタタグ整備** — AI も人間もリッチに認識するメタ情報を統一整備

### 実績

- **aupaircare.intraxjp.com**（Intrax 様）: Phase 1 + プリレンダリング完了、157ページ対応、sitemap 12→156 URL 拡張
